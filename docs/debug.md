#### 跳入跳出函数
因为每一行c代码都可能有多个函数，所以一般情况下断在一行代码上，跳入的可能并不是自己期望的函数，所以需要先跳出，然后继续跳入，直至跳入期望的函数。
### gdb
#### 普通断点
gdb可以断在某一行，或者某个函数上，使用命令`break`，或者`b`。
断在某一行，例如 
```
b test.c:100
```
断在某个函数上，例如 
```
b func
```
#### 单步调试
使用``next``命令，或者``n``。
#### 跳入跳出函数
对于gdb而言，跳入函数使用``step``命令，或``s``。跳出函数使用``finish``命令，或``fin``。
